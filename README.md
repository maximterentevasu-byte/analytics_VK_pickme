# VK Sheets Metrics Sync v4

Скрипт собирает недельную аналитику пабликов VK и записывает её в Google Sheets.

## Что изменилось в v4

- Все названия метрик в Google Sheets на русском.
- Пишутся только полные недели: понедельник — воскресенье.
- Первый запуск делает ретро-бэкфилл.
- Повторный запуск пересчитывает последнюю полную неделю и несколько предыдущих недель.
- Строки не дублируются: данные обновляются по ключу `ID группы + Неделя с`.
- Добавлены охваты через `stats.get`.
- Добавлены дельты WoW в абсолюте и процентах:
  - подписчики;
  - охват;
  - показы/просмотры сообщества;
  - ER по охвату.
- `posts_raw_vk` обновляется по ключу `ID группы + Неделя с + ID поста`.

## Переменные окружения

Смотри `.env.example`.

Минимально нужно:

```env
VK_TOKEN=vk1.a.xxxxx
VK_GROUPS=pickme_asbest,pick_me_ku
GOOGLE_SHEETS_SPREADSHEET_ID=...
GOOGLE_SERVICE_ACCOUNT_EMAIL=...
GOOGLE_PRIVATE_KEY=-----BEGIN PRIVATE KEY-----\n...\n-----END PRIVATE KEY-----\n
GOOGLE_SUMMARY_SHEET_NAME=weekly_summary
GOOGLE_POSTS_SHEET_NAME=posts_raw_vk
VK_ENABLE_STATS=true
```

## Важное про ретро с момента создания канала

VK API не всегда отдаёт дату создания сообщества отдельным полем. Поэтому по умолчанию скрипт начинает ретро с самого раннего доступного поста на стене.

Если нужна точная дата, задай вручную:

```env
VK_BACKFILL_FROM=2024-01-01
```

## Важное про подписчиков за прошлые недели

`Подписчики на конец недели, расчёт` считается обратно от текущего количества подписчиков и недельного чистого прироста (`Подписались - Отписались`). Это рабочая оценка, но она зависит от доступности `stats.get`.

```

```

- `posts_raw_vk` — посты за недели.
