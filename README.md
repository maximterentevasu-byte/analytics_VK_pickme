# VK Sheets Metrics Sync v7.2

Исправленная боевая версия.

- Не подставляет текущих подписчиков в историю без оснований.
- Если `stats.get` не отдаёт охваты/посетителей/подписки, поля остаются пустыми и пишется предупреждение.
- Добавлен столбец `Доля охвата подписчиков к подписчикам, %`.
- Добавлен `Лучший пост недели: просмотры`.
- PRO-метрики считаются по неделе, а не по лучшему посту.

Историческое число подписчиков нельзя восстановить точно, если раньше не было снимков или VK не отдаёт подписки/отписки через stats.get.
